# Django Project (Leave Management System)

Steps to run Project

1. Change the database settings
2. Run makemigrations and migrate to database
3. Create Superuser user (admin user)
	py manage.py createsuperuser
4. Create Group 'Adminstrator' in admin site
5. Add admin user with group name 'Adminstrator'
6. Create Accounts using admin credentials 
7. Leave Request are assigned based on "report to" Fields.
